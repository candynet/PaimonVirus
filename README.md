# PaimonVirus
派蒙远控病毒 -h


食用方法：
直接打开exe程序（日文语音，线性增长）

命令行参数运行：
派蒙远控病毒 [-l] [语言] [-i] [增长方式]
派蒙远控病毒 -h   显示此帮助
派蒙远控病毒 -v   显示版本信息

语言：
zh 中文语音     jp 日文语音（默认）
en 英文语音     ko 韩文语音

增长方式：
linear 线性增长（默认）
index 指数增长
random 随机增长

例如：
派蒙远控病毒 -l jp -i index     使用日文语音指数增长

两个参数可以只取其一，未规定的参数按照默认值执行！
